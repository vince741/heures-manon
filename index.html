INSTALLATION

Installer l’application

Pour utiliser AidePlanning comme une vraie application, ouvre le lien dans le navigateur puis ajoute-le à l’écran d’accueil.

Sur iPhone

Ouvre le lien dans Safari, appuie sur le bouton Partager □↑, choisis Sur l’écran d’accueil, puis appuie sur Ajouter.

Sur Android

Ouvre le lien dans Google Chrome, appuie sur le menu ⋮, puis choisis Ajouter à l’écran d’accueil ou Installer l’application.

Une fois installée, AidePlanning peut fonctionner hors ligne.
