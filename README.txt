AidePlanning — installation gratuite

1. Dézipper le fichier.
2. Créer un nouveau dépôt GitHub public.
3. Envoyer tous les fichiers à la racine.
4. Settings > Pages > Deploy from a branch > main > /root.
5. Ouvrir le lien GitHub Pages dans Chrome sur Android.
6. Menu ⋮ > Ajouter à l'écran d'accueil ou Installer l'application.
7. Ouvrir une première fois avec Internet pour activer le mode hors ligne.

Les données restent uniquement sur le téléphone.
Faire régulièrement une sauvegarde complète depuis l'onglet Résumé.
